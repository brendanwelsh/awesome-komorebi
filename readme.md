# Awesome komorebi [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

A curated list of resources for [komorebi](https://github.com/LGUG2Z/komorebi), the tiling window manager for Windows.

## Core
- [komorebi](https://github.com/LGUG2Z/komorebi) - the tiling window manager.

## Status bars
- [YASB](https://github.com/amnweb/yasb) - Yet Another Status Bar; popular komorebi pairing.

## Hotkeys
- [whkd](https://github.com/LGUG2Z/whkd) - simple hotkey daemon for komorebi.

## Theming & color
- [yasb-wallpaper-engine-color-sync](https://github.com/brendanwelsh/yasb-wallpaper-engine-color-sync) - tint the YASB bar and Windows taskbar to the active Wallpaper Engine wallpaper.

## Config & tooling
- [TranslucentTB](https://github.com/TranslucentTB/TranslucentTB) - transparent taskbar; pairs well.

## Guides
